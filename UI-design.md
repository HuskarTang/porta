# Porta UI 设计文档
本文档基于 `attachments/` 中的 UI 原型图逐一整理，转写为可实施的页面结构、组件与交互说明。

---

# 全局布局规范
- 左侧固定导航：站点 / 社区 / 服务 / 节点管理 / 服务发布 / 设置（根据页面不同高亮）
- 顶部页签导航：站点概览 / 社区管理 / 服务管理 / 设置 / 服务发布 等（视具体页面出现）
- 主内容区：卡片统计区 + 表格/表单区
- 统一元素：状态标签（在线/离线/连接中/畅通/断开）、主按钮（深色）、次按钮（浅色）

---

# 逐图说明

## Clipboard_20260115085925.png - 设置 / 节点与网络配置
**页面定位**：节点基础配置与网络参数管理。  
**布局结构**：
- 右侧主内容分为「节点配置」与「网络配置」两大卡片。
**主要组件**：
- 节点名称输入框
- 节点 ID、UUID 只读展示 + 复制按钮
- 密钥文件路径展示（只读）
- 按钮：导出密钥、导入密钥
- 网络配置：
  - 启用 TCP（开关）+ TCP 端口输入（0=系统随机）
  - 启用 QUIC（开关）+ QUIC 端口输入（0=系统随机）
  - 外网访问地址列表（表格：地址/域名、协议、端口、操作）
  - 新增外网地址行：地址输入 + 协议下拉 + 端口输入 + 添加按钮
  - 启用 mDNS 发现（开关）
  - 启用 DHT（开关）
**交互与状态**：
- 端口输入为数字，0 表示随机
- 列表行支持删除操作
- 开关切换即时生效并保存

## Clipboard_20260115090708.png - 社区管理 / 社区列表
**页面定位**：发现并加入社区节点。  
**布局结构**：
- 顶部搜索栏：输入社区 MultiAddr 或邀请码 + 刷新按钮
- 社区列表表格
**表格字段**：
- 社区名称、描述、对等节点数、状态、操作
**交互与状态**：
- 状态标签：已加入 / 未加入
- 操作按钮：查看详情 / 加入
- 刷新按钮触发列表更新

## Clipboard_20260115095639.png - 社区详情 / 可订阅服务
**页面定位**：查看社区详情并订阅社区内服务。  
**布局结构**：
- 顶部：返回社区列表、社区名称、社区描述、节点数量标签
- 内容：可订阅服务表格
**表格字段**：
- 服务名称、类型、远端端口、提供者、描述、操作
**交互与状态**：
- 操作按钮：订阅 / 已订阅（状态切换）
- 订阅后记录在本地订阅表

## Clipboard_20260115103426.png - 服务管理 / 已订阅服务列表
**页面定位**：管理已订阅并建立隧道的服务。  
**布局结构**：
- 统计卡片：总服务数 / 已连接 / 已断开
- 服务列表表格
**表格字段**：
- 服务别名、类型、社区来源、远端地址、本地映射、隧道状态、操作
**交互与状态**：
- 状态标签：畅通 / 连接中 / 断开
- 操作按钮：访问（打开本地映射地址）+ 更多操作（菜单）
- 当状态为连接中或断开时，访问按钮可置灰

---

## Clipboard_20260115143809.png - 服务访问 / WebView 访问页
**页面定位**：访问 HTTP/HTTPS 服务的内嵌浏览器页。  
**布局结构**：
- 顶部地址栏：本地映射地址 + 复制地址按钮
- 内容区：WebView/Iframe 容器（加载状态提示）
**交互与状态**：
- 加载中显示占位提示与目标 URL
- 复制地址将本地映射链接复制到剪贴板

---

## Clipboard_20260115105918.png - 服务发布 / 已发布服务列表
**页面定位**：管理本节点已发布的服务。  
**布局结构**：

## Clipboard_20260115110645.png - 服务发布 / 新增发布弹窗

**交互与状态**：
- 按钮：取消 / 确认发布
- 必填校验：服务名称、监听端口

## Clipboard_20260115114042.png - 服务映射模型图
**图示含义**：
- CommunityNode 作为中转枢纽
- 内网1 的 EdgeNode 通过服务映射提供访问入口
- 内网2 的 EdgeNode 代理目标服务
- EdgeNode 间建立隧道实现服务转发

---

## Clipboard_20260115135550.png - 安全服务映射模型图
**图示含义**：
- 至少两跳的代理链路（多个 EdgeNode / CommunityNode）
- 访问节点通过多跳代理访问远端服务
- 适用于安全加固或复杂网络场景
